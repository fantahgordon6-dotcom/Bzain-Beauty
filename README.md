# LearnEase E-Learning Website

A simple responsive static website scaffold for an e-learning platform. It includes:

- Landing page with hero section and featured courses
- Course highlights, features, testimonials, and contact form
- Dark mode toggle with a minimal JavaScript interaction

## Files

- `index.html` — main website page
- `styles.css` — responsive layout and visual styling
- `script.js` — theme toggle and contact form feedback

## Usage

1. Open `index.html` in a browser.
2. Edit content and styles in the source files.
3. Optionally host it with any static web server.

## Notes

This is a starter website structure that can be extended with additional pages, course filtering, authentication, or CMS integration.
